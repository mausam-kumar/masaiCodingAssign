Object
  • Everything in Javascript created using new Keyword is an object.
  • Threre is many way to create an object like using curly braces ,using functions or using object.create() Method.
  • We can access value inside object using dot Notation or by using square bracket.
  • Object store information in key value pair method
  • To add new key value value or to modify existing key value pair we using dot notation
    Array
  • Array is also an another kind of object and its a non-primitive Datatype.
  • There are many ways to create an Array like using square bracket or by using Array.from().
  • To add new element to an array we use push and to remove last element we use pop
  • There are many others methods available in array to modify it
  • There are also many ways available to loop an array like for loop while loop or by using for x in array or by using higher order function like map
    Functions
  • Functions are one of the fundamental building blocks in JavaScript
  • To use a function, you must define it somewhere in the scope from which you wish to call it.
  • A function definition (also called a function declaration, or function statement consists of the function keyword, followed by the name of the function
  • We can also pass various primitive and non primitive datatype as arguments to the function.
  • Threre are many ways to call function like by call(),bind() and apply() mathod
    String
  • The String object is used to represent and manipulate a sequence of characters.
  • The strings in javascript are immutable it cannot br modified.
  • There are many methods defined for string like length, indexOf, charAt strCmp
  • There are two ways to access an individual character in a string. The first is the charAt() method and second in square bracket method.
    Numbers
  • Number is a primitive wrapper object used to represent and manipulate numbers like 37 or -9.25.
  • The Number constructor contains constants and methods for working with numbers.
  • Number may also be expressed in literal forms like 0b101, 0o13, 0x0A.
  • When used as a function, Number(value) converts a string or other value to the Number type. If the value can't be converted, it returns NaN.
  • Number() Creates a new Number value.
    null
  • The value null represents the intentional absence of any object value.
  • It is one of JavaScript's primitive values and is treated as falsy for boolean operations.
  • The value null is written with a literal: null. null is not an identifier for a property of the global object, like undefined can be.
  • Instead, null expresses a lack of identification, indicating that a variable points to no object. In APIs, null is often retrieved in a place where an object can be expected but no object is relevant.
    undefined
  • The global undefined property represents the primitive value undefined
  • It is one of JavaScript's primitive types.
  • undefined is a property of the global object.
  • A variable that has not been assigned a value is of type undefined
  • A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value.
  • A function returns undefined if a value was not returned.
